# blueprint-vibe

This project intentionally mirrors the simple demo hosted at [barsdeveloper.github.io/ueblueprint](https://barsdeveloper.github.io/ueblueprint/).
It imports the `ueblueprint` web component, loads the same sample graph text into `<ueb-blueprint>`, and keeps the layout identical: a full-height blueprint preview on a plain background.
